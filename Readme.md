# Ejercicio práctico de Git

## ¿Qué es Git?

Git es un sistema de control de versiones que permite registrar los cambios realizados en un proyecto, facilitando el trabajo colaborativo y el seguimiento del historial de modificaciones.

## ¿Qué aprendí?

En este ejercicio aprendí a:

- Instalar y configurar Git.
- Inicializar un repositorio con `git init`.
- Crear archivos dentro del proyecto.
- Agregar archivos al área de preparación mediante `git add`.
- Crear commits utilizando `git commit`.
- Consultar el estado del repositorio con `git status`.
- Ver el historial de cambios con `git log`.
- Subir un proyecto a GitHub utilizando `git push`.

## Comandos utilizados

```bash
git init
git status
git add Readme.md
git commit -m "Primer commit: Añadir README"
git log
git remote add origin
git branch -M main
git push -u origin main
